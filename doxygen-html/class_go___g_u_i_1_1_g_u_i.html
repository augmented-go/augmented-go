Detailed Description

Main part of the Augmented-Go Gui. Uses the GUI.ui It connects with the backend to get, and send information. Mainly these information are the camera picture and user interactions with the gui (buttons, dialogs, etc.) The main window shows two smaller windows. One window is the augmented view. The other one the virtual view. These views can be switched by clicking the view switch button.
