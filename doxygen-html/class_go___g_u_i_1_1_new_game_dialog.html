Detailed Description

The NewGameDialog is part of Augmented-Go Gui. Uses the NewGameDialog.ui The Dialog lets the user choose several settings for a go-game listed below:

  • Name of the game
  • Name of black and white player
  • Rules: Komi, Japanese scoring and if two consecutive passes end the game When the user presses "OK" the information gets signaled and the dialog closes.
